# Ospedale

Sistema de gestión hospitalaria desarrollado en **Java (Swing)** como proyecto académico. Permite administrar pacientes, doctores, citas médicas, hospitalizaciones y recetas a través de una interfaz gráfica de escritorio.

## Ejecución

1. Abrir el proyecto en **Apache NetBeans**.
2. Compilar y ejecutar desde el IDE (`Run Project`).
